# DevClub - Genesis

サークル向けオンボーディングSaaS。Web開発サークルの新入生が離脱しないよう、GitHubのIssueをAIが極小ステップに分解し、成長実感・貢献実感・没頭感を提供します。

## Getting Started

```bash
npm install
cp .env.example .env.local  # 環境変数を設定
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS
- Firebase (Auth + Firestore)
- Gemini API
- GitHub API
